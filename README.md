SuperStore Sales Analysis 📊
A data-driven project analyzing retail sales performance using Power BI for interactive dashboards and Python for automated data insights.

🎯 Key Objectives
Analyze Sales & Profit across different categories.

Track Monthly Trends to identify seasonality.

Evaluate Regional Performance and payment methods.

🛠️ Tech Stack
Python (Pandas, Matplotlib)

Power BI (.pbix)

VS Code
